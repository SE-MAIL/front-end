# front-end
This is the application code of our service "sinabro".

#stack
- react-native
- react-native hook
- react-native context
- react-native navigation
- axios

#how to run
## install dependency
yarn
```
yarn add
```
npm
```
npm install
```
in IOS
```
cd ios
pod install
```
### start app
in IOS
```
react-native run-ios
```
in android
```
react-native run-android
```
